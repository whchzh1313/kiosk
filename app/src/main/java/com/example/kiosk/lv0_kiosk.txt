요구사항1: 메인 메뉴판 화면
: 현재 주문할 금액 입력 [0]
: 메뉴 선택시 -> 상세 메뉴화면으로 이동
: 잘못된 번호 선택 시 -> 예외처리
: 프로그램 종료을 위한 -> 번호 정의 [0]
요구사항2: 상세 메뉴 화면
: 상세 메뉴에서 금액 출력
: 선택 시 장바구니에 메뉴추가후 메인 메뉴로 진입
: 이전 메뉴로 돌아가기 위한 -> 번호 정의 [0]
요구사항3: 결제 화면
: 선택하신 메뉴의 목록과 가격 출력
: 메뉴의 총 금액 출력
: 입력을 통한 [0]취소하기 [1]주문하기
: 취소시 장바구니에 담은 메뉴 삭제할지 체크 [0]삭제하기 [1]더 주문하기
요구사항4: 결제 완료 후 화면
: 종료와 추가주문을 구분
